# CLERK

A command-line tool for situational awareness and context-sensitive operations.

## Overview

CLERK (Command-Line Executor for Rapid Knowledge) is a pivotal component of the PHOTON platform. It provides context-sensitive operations based on the current working directory and its contents.

## Index

-   `app.py`: The main Textual application setup.
-   `clerk.py`: Core logic and role definitions for the CLERK tool.
