clerk.logger.app.find_git_home_directory()[source]¶

Find the Git home directory starting from the current directory. Returns the path to the Git home directory as a Path object if found, otherwise returns None.

clerk.logger.app.get_docsrc_directory()[source]¶

Check if a ‘docsrc’ directory exists in the Git home directory. If it exists, returns the full path to ‘docsrc’, otherwise returns None. Additionally, sets the current working directory to ‘docsrc’ if it exists.
