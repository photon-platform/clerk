usage¶

init

The init command creates new Python projects using standard templates.

build

The build command shapes Python modules, classes, and functions.

curate

The curate command: - branches: Lists branches. - create-release-branch: Creates a release branch (e.g. release-0.1.0). - merge-to-main: Merges a branch to main. - create-tag: Creates a git tag. - rollup: Automates the release process (version bump, changelog, merge, tag). - status: Shows git status and diff since branching.

gather

The gather command scrapes and consolidates content from URLs and repositories.

log

The log command creates standardized log entries in the project’s documentation.
